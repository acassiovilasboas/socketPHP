Sistema Bancário
Abrir Conta

Serviço para abertura de contas para pessoas físicas.

Dados obrigatórios: CPF

Sacar

Serviço para saque em contas.

Dados obrigatórios: CPF ou Número da Conta

Depositar

Serviço para depositar recuros em contas.

Dados obrigatórios: CPF ou Número da Conta

Transferir

Serviço para transferir recuros entre contas.

Dados obrigatórios: CPF ou Número da Conta Origem e Destino

Empréstimo

Serviço para captar recuros.

Dados obrigatórios: CPF ou Número da Conta e Valor do Empréstimo
